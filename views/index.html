3Ô∏è‚É£ Manage Fun Facts

GET /states/{stateCode}/funfact - Retrieves a random fun fact.

POST /states/{stateCode}/funfact - Adds a fun fact.

PATCH /states/{stateCode}/funfact - Updates a fun fact at a given index.

DELETE /states/{stateCode}/funfact - Removes a fun fact at a given index.
